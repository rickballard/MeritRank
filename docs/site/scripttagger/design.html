ScripTagger: design notes

Behavior-safe, jargonless UX

  • One big action: “Record / Import.”
  • Five simple chips: Truth · Evidence · Conduct · Anti-Manipulation · Amplification.
  • + Source link is always visible before any critique.
  • Kid-friendly text: “Was this part honest?” “Did it cite where facts came from?”
  • Private by default: sessions save locally; sharing is opt-in.

Unverified → Consensus → Positive-only publication

Each submission is an observation. Observations for the same event are clustered by media URL, timestamps, and audio/text fingerprint. We compute an internal consensus; only well-supported positive moments become public RepTag credits. Negatives never publish.

score(dim, segment) = weighted_mean_i ( user_weight_i × evidence_weight_i × rating_i )

user_weight = f(RepTag⁺)          # positive-only component, bounded
evidence_weight = g(citations, independence, cross-agreement)
publish if: support ≥ threshold && independence ≥ min && no-policy-violations
  → mint positive credit (RepTag)

Consent & safety

  • “Assistance, not verdicts” banner on every page.
  • Opt-in “Contribute my session” toggle; JSON is signed & hash-anchored.
  • Minimize PII; redact minors and sensitive data where possible.
