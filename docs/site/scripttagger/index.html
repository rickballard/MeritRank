ScripTagger (pre-alpha)

Tag live talks and videos in ~20-sentence windows for TruthAlignment, EvidenceHygiene, CivicConduct, Anti-Manipulation, and AmplificationQuality.

How it works

  1. Ingest speech/text (paste transcript; mic & audio later).
  2. Segment into rolling ~20-sentence chunks.
  3. Tag each chunk with scores + short rationales + links.
  4. Export a Markdown report (plus JSON with all details).
  5. Publish (optional) a prefilled evidence issue to MeritRank.

Why it’s needed

  • Turns chaotic streams into auditable evidence.
  • Encourages careful sourcing and correction in-the-moment.
  • Feeds RepTag with positive recognition by design.

Data & consent

  • Local-first by default (sessions save to your device).
  • Opt-in to contribute signed session JSON (hash-anchored) to an append-only log.
  • Clear “assistance, not verdicts” language; unverified claims are labeled.
