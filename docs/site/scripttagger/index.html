ScripTagger (pre-alpha)

One-tap capture → simple, shareable summary. Transcript optional.

What you get by default

  • Two-panel summary: Best claims vs Questions to verify.
  • Tap to add sources (links/screenshots). No jargon.
  • Transcript available, but off by default.

Why

  • Most people won’t read transcripts right after an event; a short digest helps counter hype.
  • Public events are usually posted online; we can infer the speaker from the URL + metadata.
  • Feeds RepTag using positive-only publication; negatives stay private for internal separation.
